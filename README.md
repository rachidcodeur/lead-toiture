# Lead Toiture - Landing page artisans couvreurs

Landing page statique optimisée pour le SEO destinée aux artisans couvreurs à la recherche de demandes de devis qualifiées.

## Structure

- `index.html` : structure HTML sémantique, métadonnées SEO, contenu marketing.
- `styles.css` : design system complet (couleurs, typo, layout, cards, boutons, responsive).
- `script.js` : animations légères (apparition au scroll, année dynamique, scroll ciblé).

## Stack

- HTML5
- CSS3 (sans framework, design system custom)
- JavaScript vanilla (sans dépendance externe)

## Lancer le site en local

Ouvrez simplement `index.html` dans votre navigateur ou servez le dossier via un petit serveur statique (ex. `npx serve .`).
